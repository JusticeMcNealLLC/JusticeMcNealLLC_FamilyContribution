How to Export from Fidelity

  1. Log in to Fidelity NetBenefits or Fidelity.com
  2. Navigate to Positions (under Portfolio)
  3. Click the Download icon (arrow down) in the top-right of the positions table
  4. Select CSV format and download
  5. Upload that file below, or enter holdings manually

The CSV should contain columns: Symbol, Description, Quantity, Last Price, Current Value
